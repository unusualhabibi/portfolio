# Strategic Copywriter Portfolio (Static Site)

A modern, minimal, production-ready personal portfolio website built with plain **HTML5**, **CSS3**, and **Vanilla JavaScript**.

Designed for a strategic copywriter and digital creator with a premium editorial style inspired by high-end agency websites.

## Project Structure

```text
/portfolio
  index.html
  styles.css
  script.js
  README.md
```

## Features

- Sticky navigation bar
- Scroll progress indicator
- Smooth scrolling navigation
- Animated hover cards
- Scroll reveal animations
- Interactive copy test section
- Mobile responsive layout

## Design System

- Background: `#FAFAF9`
- Primary text: `#111110`
- Accent: `#000`
- Max content width: `1100px`
- Typography: Inter + system fallbacks

## Sections Included

1. Hero
2. Services
3. Copywriting Autopsy
4. Interactive Copy Test
5. Testimonials
6. Final CTA

## Run Locally

No build tools required.

1. Clone or download this folder.
2. Open `index.html` directly in your browser.

## Deploy

### GitHub Pages

1. Push this folder to a GitHub repository.
2. In **Settings → Pages**, set source to the root branch.
3. Save and wait for deployment.

### Netlify

1. Drag-and-drop the folder in Netlify Drop, or connect the repository.
2. Build command: *(none)*
3. Publish directory: `/`

## Notes

- The site is fully static and framework-free.
- JavaScript powers scroll progress, reveal animations, smooth navigation, and interactive CTA testing.
